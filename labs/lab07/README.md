# Лабораторна робота №7: Linear SVM, char-ngrams та робота з дисбалансом

1. **Підзадача класифікації:** Виявлення фейкових новин (True/False) в умовах дисбалансу класів (False ~ 23.3%).
2. **Baseline з ЛР6:** Logistic Regression + TF-IDF word(1,2) (Macro-F1: 0.8510).
3. **Перевірені SVM-варіанти:** LinearSVC з word n-grams (1,2) та LinearSVC з char_wb n-grams (3,5). Обидві моделі використовували `class_weight='balanced'`.
4. **Робота з imbalance:** Дисбаланс компенсувався зважуванням класів, що дозволило моделі не ігнорувати міноритарний клас (фейки).
5. **Вибір порогу:** Під час аналізу PR-кривої на validation set обрано поріг `0.0`, оскільки він дає найвищий F1-Score для міноритарного класу. Зсув порогу до `0.2` розглядається як альтернатива (recall-first), якщо бізнес-вимога — не пропускати жодного фейку ціною хибних спрацювань.
6. **Найкраща модель:** LinearSVC + TF-IDF char_wb(3,5). Вона показала найкращі результати на тестовій вибірці (Test Accuracy: 0.9236, Test Macro-F1: 0.8903).
7. **Що робити далі:** Аналіз помилок показав, що модель плутається у випадках прямого цитування пропаганди (noisy labels) та майстерної імітації офіційного стилю (overlap). Для вирішення цього потрібен перехід до семантичних моделей (Embeddings / Transformers), які здатні розуміти ширший контекст речення, а не лише статистику символів.
