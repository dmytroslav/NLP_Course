
### Стислий звіт:
1. **Корпус:** Telegram-новини (10.7к повідомлень)
2. **Моделі:** Word2Vec та FastText (Skip-Gram, size=100)
3. **Аналіз:** Протестовано 10 слів різних типів та 5 доменних термінів
4. **Ключовий інсайт:** Word2Vec краще вловлює контекст подій (що куди влучило), тоді як FastText фокусується на морфології слова
5. **Вердикт:** Embeddings корисні для розуміння структури словника, але FastText потребує обережного використання на рідкісних словах через ризик "буквеного шуму"
