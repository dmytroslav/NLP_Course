# Нотатки щодо Word Embeddings (ЛР9)

### 1. Який корпус
Використано корпус новинних Telegram-каналів, підготовлений у попередніх лабораторних роботах.
* **Кількість документів:** 10,734
* **Кількість токенів:** ~148,068
* **Тип тексту:** Очищений текст (поле `text`), токенізований за допомогою `simple_preprocess` (нижній регістр, видалення пунктуації).

### 2. Які моделі
Натреновано дві архітектури з бібліотеки `gensim`:
1. **Word2Vec** (Skip-Gram)
2. **FastText** (Skip-Gram)

### 3. Параметри тренування
Для чесного порівняння обох моделей використано однакові параметри:
* `vector_size = 100`
* `window = 5`
* `min_count = 3`
* `sg = 1` (Skip-Gram обрано для кращої роботи з рідкісними та доменними словами у невеликому корпусі).

### 4. 10 слів для Nearest Neighbors
Аналіз проводився для таких слів: *україна, ракетний, тривога, зсу, ппо, блекаут, русня, бахмут, допомога, повідомляють*.

### 5. 5 доменних термінів
| Термін | Модель-переможець | Чому? |
| :--- | :--- | :--- |
| **ракетний** | Word2Vec | Знайшов семантичні зв'язки (удар, міст, будинок), а не просто словоформи. |
| **тривога** | Word2Vec | Чітко виділив контекст "повітряна тривога оголошена". |
| **зсу** | Word2Vec | Знайшов дієслова дії (знищили, звільнили, окупантів). |
| **бахмут** | FastText | Знайшов географічно близький "Часів Яр" через підслова (n-grams). |
| **допомога** | Обидві | Дали гарний результат: W2V (фінансування), FT (млрд, долар). |

### 6. 5 кейсів "корисно / некорисно"
1. **ракетний (Useful):** W2V показав об'єкти ударів (міст, будинок), що корисно для аналізу подій.
2. **повідомляють (Useful):** Обидві моделі знайшли функціональні синоніми (пишуть, джерела), корисні для стилістичного аналізу.
3. **бахмут (Partly):** Змішаний результат. FT знайшов "Яр", але додав "дтп" через шум у підсловах.
4. **блекаут (Weak):** W2V видав OOV, а FT згенерував випадкових людей через брак даних.
5. **русня (Weak):** Провал FastText — згрупував назви місяців ("липня", "червня") суто за закінченням "-ня".

### 7. Порівняння Word2Vec vs FastText
* **Word2Vec** краще знаходить семантичні та функціональні синоніми для частих слів.
* **FastText** виграє у роботі з морфологією (бачить усі відмінки слова "Україна"), але сильно "шумить" на рідкісних словах через n-грами.

### 8. Головний висновок
Embeddings дають корисний сигнал для аналізу тематики та стилістики (наприклад, шаблони повідомлень про тривоги). Проте для рідкісної доменної лексики корпус у 150к токенів є замалим для стабільних результатів.
