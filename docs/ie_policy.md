# Information Extraction Policy (Lab 4)

## 1. DATE (Дати)
- **Що вважаємо:** Точні календарні дати у форматі DD.MM.YYYY.
- **Формат на виході:** YYYY-MM-DD (ISO формат).
- **Не вважаємо:** Роки без місяця ("в 2022 році"), відносні дати ("вчора").
- **Edge cases:** Відсутність нулів ("1.2.2022" - не беремо для високої точності precision-first).

## 2. AMOUNT (Суми)
- **Що вважаємо:** Грошові суми з вказанням валюти.
- **Формат на виході:** Число + Множник + Нормалізована валюта (UAH/USD/EUR).
- **Не вважаємо:** Відсотки ("100%"), просто числа без валюти ("втратили 50 танків").
- **Edge cases:** Розділювачі ("10,5 тис грн" vs "10.5 тис грн").

## 3. LOCATION (Локації)
- **Що вважаємо:** Назви міст та регіонів України з нашого закритого словника (`locations_ua.txt`).
- **Формат на виході:** Назва міста у називному відмінку з великої літери (Київ).
- **Не вважаємо:** Назви вулиць, країни (крім наявних у словнику).
- **Edge cases:** Збіг зі звичайними словами (якщо такі є), відмінки (поки шукаємо базову форму для високого precision).
