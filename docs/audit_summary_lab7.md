# Audit Summary: Lab 7 (Linear SVM + char-ngrams)

1. **Назва задачі:** Бінарна класифікація українських новин на предмет дезінформації та пропаганди (True/False).
2. **Baseline із ЛР6:** TF-IDF word(1,2) + Logistic Regression (class_weight='balanced').
3. **Протестовані SVM-моделі:**
   - LinearSVC + TF-IDF word(1,2)
   - LinearSVC + TF-IDF char_wb(3,5)
4. **Найкращий результат:** LinearSVC + TF-IDF char_wb(3,5) (Accuracy: 0.9236, Macro-F1: 0.8903).
5. **Вплив char-ngrams:** Використання символьних n-грам значно покращило результати (Macro-F1 зріс з 0.8750 до 0.8903). Це дозволило моделі краще справлятися з варіативним написанням, друкарськими помилками та специфічним шумом з Telegram-каналів.
6. **Вплив class_weight:** Параметр `class_weight='balanced'` допоміг зберегти стабільний Recall для міноритарного класу (фейків).
7. **Вибір порогу (Threshold):** За результатами аналізу PR-кривої на валідаційній вибірці було обрано дефолтний поріг `0.0`. Він забезпечує найкращий баланс (F1-Score: 0.8434 для міноритарного класу) між Precision (0.88) та Recall (0.80). Для більш агресивної фільтрації фейків у майбутньому можна зсунути поріг до `0.2` (Recall: 0.85).
8. **Найчастіші типи помилок:**
   - *Overlap класів:* Фейки маскуються під офіційні новинні зведення, використовуючи лексику реальних новин (наприклад, згадки державних органів).
   - *Noisy labels:* Реальні новини хибно класифікуються як фейки через пряме цитування російської пропаганди чи офіційних осіб РФ.
