# Audit Summary: Laboratory Work #9

**Дата:** 2026-04-17  
**Об'єкт аудиту:** Word2Vec та FastText моделі на корпусі Telegram-новин.

### 1. Огляд даних
* **Джерело:** Новинні Telegram-канали про війну в Україні.
* [cite_start]**Обсяг:** 10,734 документи, ~148к токенів
* **Препроцесинг:** Видалення пунктуації, нижній регістр, фільтрація коротких текстів (<3 слів).

### 2. Результати аналізу Neighbors
* **Найсильніші приклади (Semantic consistency):**
    * `повідомляють` -> `пишуть`, `джерела`, `інформацію` (W2V).
    * [cite_start]`ракетний` -> `удар`, `будинку`, `мосту` (W2V)
* **Найслабші приклади (Noise/Randomness):**
    * `русня` -> `липня`, `червня` (FT — помилка за закінченням).
    * [cite_start]`блекаут` -> `патрушев`, `патріархату` (FT — галюцинації n-грамів)

### 3. Доменна специфіка
Доменні терміни (*ракетний, тривога, ЗСУ*) отримали стабільні представлення у Word2Vec. [cite_start]FastText виявився корисним для розширення словника (пошуку відмінків), але менш придатним для виявлення нових контекстів на цьому обсязі даних.

### 4. Висновок щодо використання
Embeddings **варті використання** для EDA (дослідницького аналізу), групування схожих за стилем новин та покращення пошуку за ключовими словами. [cite_start]Для задач класифікації (ЛР6-7) вони можуть дати приріст, якщо використовувати попередньо натреновані великі моделі або розширити поточний корпус.
