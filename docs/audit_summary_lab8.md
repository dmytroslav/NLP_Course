# Audit Summary: Lab 8 (Topic Modeling)

1. **Розмір корпусу після фільтрації:** 10734 документи (відкинуто 1 надто короткий текст)[cite: 245].
2. **Протестовані моделі:** LSA (TruncatedSVD + TF-IDF) та LDA (LatentDirichletAllocation + CountVectorizer)[cite: 246].
3. **Протестовані значення k:** 5 та 8 тем для кожної моделі.
4. **Найкращі теми (LSA, k=8):** "Ракетні та артилерійські удари" (Тема 4) та "Оголошення повітряних тривог" (Тема 6). Вони мають чітко виражений семантичний центр та релевантні топ-документи.
5. **Найгірші теми:** Тема 0 (Stop-word topic) та Тема 7 (Mixed topic)[cite: 249].
6. **Що їх зіпсувало:** Брак якісної фільтрації українських стоп-слів на етапі векторизації та використання стемінгу/початкових форм замість повноцінної лематизації. Часті слова перетягнули на себе вагу алгоритму.
7. **Краща модель для кейсу:** LSA. Вона краще впоралася з дисбалансом частот завдяки метриці TF-IDF, тоді як LDA "потонула" в загальній новинній лексиці та прийменниках.
8. **Подальші кроки:** Додати кастомний словник українських стоп-слів, переключитися на використання `lemma_text` та протестувати виділення тем на основі біграм (ngram_range=(1,2)), щоб ловити сталі вирази.
